## Install.sh
```bash
curl -fsSL https://raw.githubusercontent.com/Tapi-Mandy/guhShot/main/install.sh | bash
```

## Keybinds
### mangowc / guhwm:
```ini
bind=NONE, Print, spawn, guhshot --full
bind=SHIFT, Print, spawn, guhshot --select
```
